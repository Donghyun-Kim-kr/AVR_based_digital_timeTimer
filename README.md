# AVR_based_digital_timeTimer
AVR (Atmega128a) based digital timeTimer style firmware &amp; device

## Requirements 
- Microchip studio (Atmel studio 7.0)
- KD128-pro (atmega128a) board
- 1 piezo buzzer
- 1 variable register
- 8 button switch for External Interrupts
